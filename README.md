# damianbiniarz.com

Personal landing page with app catalog and animations.

## Stack
- Vanilla HTML/CSS/JS (no framework — fast, simple deploy)
- GSAP for scroll animations
- Cloudflare Pages hosting

## Structure
```
index.html    - Main page
styles.css    - Styles (dark theme)
app.js        - Animations & interactions
assets/       - Images, icons
```

## Sections
1. Hero (name, tagline, CTA)
2. About (short bio)
3. Apps catalog (HomeFresh, Dive, Gym...)
4. Contact / Social links

## Deploy
Push to GitHub → Cloudflare Pages auto-deploy
